AdFlow Pro 🚀

> A production-style moderated ads marketplace where clients submit listings, moderators review content, administrators verify payments, and only approved ads become visible for a limited package-based duration.

🌐 Live Demo

👉 [https://alveena-adflow-pro.vercel.app](https://alveena-adflow-pro.vercel.app)

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Next.js 16, React 19, TypeScript |
| Backend | Node.js, Express.js |
| Database | Supabase (PostgreSQL) |
| Authentication | JWT + bcryptjs |
| Validation | Zod |
| Deployment | Vercel (Frontend) |

## 👥 User Roles

- **Client** — Register, submit ads, track status, submit payment proof
- **Moderator** — Review submitted ads, approve or reject content
- **Admin** — Verify payments, publish ads, manage users
- **Super Admin** — Full system access
